# NBA Player Longevity Prediction (Feature Engineering)

## 📌 Project Overview
This project focuses on feature engineering for predicting NBA player career longevity (5+ years).

## ⚙️ Key Steps
- Data cleaning and preprocessing
- Removal of non-predictive features
- Correlation analysis and multicollinearity reduction
- Feature engineering (efficiency metrics)
- Handling missing values

## 📊 Engineered Features
- Points per minute
- Player efficiency rating

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Seaborn / Matplotlib

## 🎯 Goal
To prepare a clean, model-ready dataset for machine learning prediction.
